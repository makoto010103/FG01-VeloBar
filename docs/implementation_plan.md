# VBTデバイス開発 実装計画書

## ゴール
Seeed XIAO nRF52840 と ICM-42688 センサーを使用したVBTデバイスを開発し、Webブラウザ上でリアルタイムに挙上速度を確認・グラフ化できるシステムを構築する。

## 構成
- **ファームウェア**: Arduino IDE (C++)
- **アプリ**: Web Bluetooth API (HTML/JavaScript)

## 詳細設計
### 1. ハードウェア接続
- **XIAO nRF52840** <-> **ICM-42688**
    - 3V3 -> VCC
    - GND -> GND
    - SDA (D4) -> SDA
    - SCL (D5) -> SCL

### 2. ファームウェア
- 姿勢推定 (Madgwickフィルター) で重力成分を除去
- 鉛直方向の加速度を積分して速度を算出
- BLE通知で速度データを送信

### 3. Webアプリ
- `Chart.js` を使用して速度の推移をグラフ化
- 音声によるフィードバック機能（予定）

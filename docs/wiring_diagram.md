# VBTデバイス 配線対応表

このドキュメントは、Seeed XIAO nRF52840 と ICM-42688 (Strawberry Linux製モジュール) の正確な配線を記録したものです。

## 接続一覧

| ICM-42688 (センサー側) | XIAO nRF52840 | 役割 | プログラムでの制御 |
| :--- | :--- | :--- | :--- |
| **1 (VDD)** | **3V3** | メイン電源 | 固定(3.3V) |
| **2 (VDDIO)** | **D1** | I/O電源 | `HIGH` を出力して供給 |
| **3 (SCL)** | **D5** | I2Cクロック | 通信で使用 |
| **4 (SDA)** | **D4** | I2Cデータ | 通信で使用 |
| **5 (AD0)** | **D2** | アドレス設定 | `LOW` で 0x68 に固定 |
| **6 (~CS)** | **D3** | モード設定 | `HIGH` で I2Cモード有効 |
| **10 (GND)** | **GND** | 地面 | 固定(0V) |

## 配線のポイント

- **安定化**: 通信が不安定な場合は、`2 (VDDIO)` を `3V3` へ、`5 (AD0)` を `GND` へ直接繋ぎ変えることで、より強力に安定させることができます。
- **接触**: スポーツの動き（衝撃）に耐えるため、最終的にはハンダ付けするか、コネクタ部分をテープなどでしっかり固定することを強く推奨します。
